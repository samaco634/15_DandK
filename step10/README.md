# minikube 이용시 문제 회피 방법

이에 따라 예제 코드 job-initiator.py에서 minikube의 IP주소를 수정해야 한다.

다음 방법으로 IP주소를 확인할 수 있다. 


~~~
$ minikube ip

~~~
